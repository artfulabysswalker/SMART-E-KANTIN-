# SMART-E-KANTIN-

Aplikasi mobile untuk memudahkan proses pemesanan makanan dan minuman di kantin kampus dengan fitur keranjang belanja dan diskon berdasarkan NIM mahasiswa.

## Fitur Utama

- **Keranjang Belanja**: Pengguna dapat menambahkan berbagai item ke dalam keranjang belanja mereka sebelum melakukan pembayaran.
- **Diskon Berdasarkan NIM**: Sistem diskon otomatis berdasarkan digit terakhir NIM mahasiswa.
- **Integrasi Firebase**: Penggunaan Firebase Authentication untuk manajemen pengguna dan Firestore untuk penyimpanan data transaksi.
- **Antarmuka Pengguna yang Responsif**: Desain UI yang intuitif dan mudah digunakan pada perangkat mobile.
